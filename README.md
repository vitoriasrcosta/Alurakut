<h1 align="center"> Alurakut </h1>

<h2 align="left"> O que é o Alurakut? </h2>
<p align="left"> O Alurakut é uma rede social baseada no antigo Orkut, desenvolvida durante a semana de imersão em React do Alura. </p>
